les mots suivent l'usage du Québec (`courriel`, `arobas`) et la nouvelle orthographe (`aout`, `s'il vous plait`, sans accent circonflexe).

## découvrez

les trois questions d'ouverture de l'épisode:

- `Quel est votre nom de famille?`
- `D'où venez-vous?`
- `Quelle est votre date de naissance?`

une carte d'identité (`fiche d'identité`) liste cinq champs:

| champ               | exemple       |
| ------------------- | ------------- |
| `Nom`               | `Desjardins`  |
| `Prénom`            | `Nancy`       |
| `Date de naissance` | `1980-03-15`  |
| `Profession`        | `professeure` |
| `Nationalité`       | `canadienne`  |

## donner ses informations personnelles

### l'âge

l'âge se dit avec le verbe `avoir`, pas `être`. le français compte les années comme une possession: on _a_ un âge.

- `J'ai 4 ans.`
- `J'ai 21 ans.`
- `J'ai 40 ans.`
- `J'ai 84 ans.`

le mot `ans` est obligatoire. `J'ai 21.` est faux; il faut `J'ai 21 ans.`

> [!warning] le piège anglais
> l'anglais dit _I am 21_ (être) ; le français dit `J'ai 21 ans` (avoir + `ans`). traduire mot à mot avec `être` est l'erreur classique.

la question:

| registre | question              |
| -------- | --------------------- |
| formel   | `Quel âge avez-vous?` |
| informel | `Quel âge as-tu?`     |

### les nombres

orientation rapide pour cet épisode. le détail complet (0–69, les dizaines irrégulières `soixante-dix`/`quatre-vingts`/`quatre-vingt-dix`, la prononciation) vit dans [[fr/les-nombres]].

- `0–10` : `zéro, un, deux, trois, quatre, cinq, six, sept, huit, neuf, dix`.
- dizaines : `dix (10)`, `vingt (20)`, `trente (30)`, `quarante (40)`, `cinquante (50)`, `soixante (60)`.
- `cent` = `100`, `deux-cents` = `200` (trait d'union, orthographe rectifiée).
- `mille` = `1 000` (invariable), `un million` = `1 000 000`.

### la date de naissance

format **`AAAA-MM-JJ`** (année, mois, jour), l'ordre administratif québécois.

| dit                | écrit        |
| ------------------ | ------------ |
| `6 mars 1980`      | `1980-03-06` |
| `13 novembre 1965` | `1965-11-13` |

les mois de l'année s'écrivent en minuscule, sans majuscule:

`janvier`, `février`, `mars`, `avril`, `mai`, `juin`, `juillet`, `aout`, `septembre`, `octobre`, `novembre`, `décembre`.

> [!note] orthographe rectifiée
> `aout` s'écrit sans accent circonflexe (l'ancienne graphie `août` reste tolérée). même logique pour `s'il vous plait` plus bas.

#### la date et le chiffre « 1 »

dans une date, le `1` se dit `premier` et s'écrit `1er`. tous les autres jours prennent le nombre cardinal ordinaire.

- `le 1er mars 1967` se lit `le premier mars 1967`.
- `le 2` se lit `le deux` ; `le 13` se lit `le treize`.

## le numéro de téléphone

chaque chiffre se prononce séparément, un par un. l'indicatif régional (`514`, `450`, …) précède le numéro.

- `514 525-7530` se lit chiffre par chiffre : `cinq, un, quatre, cinq, deux, cinq, sept, cinq, trois, zéro`.

## le courriel

| symbole | se dit                       |
| ------- | ---------------------------- |
| `@`     | `arobas` (ou `a commercial`) |
| `.`     | `point`                      |

au Québec, un email s'appelle un `courriel` (mot-valise de `courrier` + `électronique`). exemple: `mariet@editionsmd.ca` se lit `mariet arobas editionsmd point ca`.

## poser des questions : informel vs formel

le manuel oppose deux registres qui gardent tous les deux le `vous` de politesse. l'informel relâche la syntaxe (`c'est quoi`, ordre sujet-verbe, pas d'inversion) ; le formel passe par l'inversion (`quel est`, `avez-vous`). le `tu` vraiment familier viendra plus tard.

| information       | situation informelle                | situation formelle                                        |
| ----------------- | ----------------------------------- | --------------------------------------------------------- |
| nom / code postal | `C'est quoi votre nom/code postal?` | `Quel est votre nom/code postal/état civil?`              |
| téléphone         | `Votre numéro de téléphone?`        | `Quelle est votre adresse/date de naissance/nationalité?` |
| âge               | `Vous avez quel âge?`               | `Quel âge avez-vous?`                                     |
| identité          | `Vous vous appelez comment?`        | `Comment vous appelez-vous?`                              |

> [!note] faits d'ici
> `Ailleurs, autrement?` (`Dire son âge : oui ou non?`). le manuel demande : _dans votre culture, dire son âge, c'est normal? est-ce impoli de demander son âge à une personne?_ au Québec la question reste sensible selon le contexte.

## ne pas comprendre

les expressions pour indiquer qu'on ne comprend pas:

- `Je ne comprends pas.`
- `Pardon?`
- `Pouvez-vous répéter, s'il vous plait?`
- `Pouvez-vous parler lentement, s'il vous plait?`
- `Hein?` (familier, à l'oral seulement)

> [!warning] registre de `Hein?`
> `Hein?` est familier, réservé à l'oral dans une situation informelle. dans un contexte formel, utiliser `Pardon?` ou `Pouvez-vous répéter, s'il vous plait?`.

## remplir un formulaire

### titres de civilité

| abréviation | titre          | pour                       |
| ----------- | -------------- | -------------------------- |
| `M.`        | `Monsieur`     | un homme                   |
| `Mme`       | `Madame`       | une femme                  |
| `Mlle`      | `Mademoiselle` | une jeune femme non mariée |

### l'adresse : abréviations courantes

| abréviation | mot complet   |
| ----------- | ------------- |
| `app.`      | `appartement` |
| `av.`       | `avenue`      |
| `boul.`     | `boulevard`   |
| `tél.`      | `téléphone`   |

### les champs du formulaire

une `fiche d'inscription` type demande:

- `Nom de famille`
- `Prénom`
- `Date de naissance` (`AAAA-MM-JJ`)
- `Adresse` (`No, rue, app.`)
- `Ville`
- `Code postal`
- `Courriel`

exemple rempli: `Hélène Ferron`, `H2P 4T8`, `mariet@editionsmd.ca`.

## à retenir

- l'âge : `avoir` + nombre + `ans` obligatoire.
- la date de naissance : `AAAA-MM-JJ`, mois en minuscule, `aout` sans circonflexe.
- le `1` d'une date se dit `premier`, s'écrit `1er` ; les autres jours sont cardinaux.
- `@` = `arobas` ; un email = un `courriel`.
- `vous` pour le formel, `tu` pour l'informel.
