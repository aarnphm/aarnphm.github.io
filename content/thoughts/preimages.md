also known as a _inverse image_, of a subset $B$ uonder a function $f: X \to Y$ is the set of all elements in the domain $X$ that maps to elements in $B$

if $f$ maps set $X$ (domain) to set $Y$ (codomain), then $B \subseteq Y$ is ::always:: a subset of $X$

| properties   | notation                                      |
| ------------ | --------------------------------------------- |
| intersection | $f^{-1}(A \cap B) = f^{-1}(A) \cap f^{-1}(B)$ |
| union        | $f^{-1}(A\cup B) = f^{-1}(A) \cup f^{-1}(B)$  |
| complement   | $f^{-1}(B^{C}) = (f^{-1}(B))^{C}$             |

A distinction to be made between _image_ and _preimage_:

> ::image of a set:: we go from _left to right_. For $A \subseteq X\;, f(A) = \{f(x) \mid x \in A\}$ is the set of outputs generated by a set of inputs
> <br/><br/>
> ::preimage of a set{h3}:: we go from _right to left_. For $B \subseteq Y\;, f^{-1}(B) = \{x\in X \mid\;f(x) \in B\}$ is the set of inputs that generated a set of outputs
