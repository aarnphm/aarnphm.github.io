_Lexical analysis, syntax analysis, type checking; syntax-directed translation, attribute grammars; compiler structure; implications of computer architecture; mapping of programming language concepts; code generation and optimization._

Midterm: Tuesday, 24 Feb 2026, 2:30 - 3:20 (class time), CNH 104

- Assignments 28%
  - There are 11 weekly assignments; each is introduced in the lab session for which you are registered.
  - Each assignment is worth 3%; the marks are summed up and capped at 28%. That is, if you don't do perfectly on some assignments, you can stil get the full 28% of the assignments.
  - If you MSAF an assignment, you get a 3-day extension.
- Late submissions are not accepted.
- midterm test 22%
- final exam 50%
  - The tests include programming questions with Jupyter. You need to bring a laptop computer that runs a web browser to the tests.
  - The assignments, the midterm test, and the final exam are to be submitted on the JupyterHub server.
